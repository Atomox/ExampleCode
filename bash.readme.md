


## Install Stupp

### apt-get

`apt-get update`
`apt-get install [package]`


### brew (OSX)
`brew update`
`brew install [package]`


### Copy to a remote server:

`scp local_file user@remotehost:/path/to/target`

### User Stuff

Add user to a group
`sudo usermod -aG group_name user_name`

Add a user:
`useradd name`

Show all members of a group:

`cat /etc/group`

Show all users

`cat /etc/passwd`
